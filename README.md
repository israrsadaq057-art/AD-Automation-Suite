## 🚀 AD Automation Suite Project

### Project Overview

This project is a **complete Active Directory Automation Suite** built for **Resource Academia International**, an educational institution in Islamabad, Pakistan. It demonstrates enterprise-level IT management capabilities.

### What I Built

| Feature | Description |
|---------|-------------|
| **Active Directory Structure** | Complete AD hierarchy with 6 departments, 100+ users, and 12 security groups |
| **User Lifecycle Management** | PowerShell scripts for onboarding, offboarding, and bulk user import |
| **Asset Tracking** | Laptop, monitor, and phone assignments with serial numbers and warranty tracking |
| **Attendance Monitoring** | Track present, absent, leave, and late days with attendance percentages |
| **Security Groups** | 12 groups with proper access control across all departments |
| **Interactive Dashboard** | HTML/CSS/JS dashboard with clickable employee profiles |
| **PowerShell Automation** | 10+ scripts for AD management, GPO automation, and backup |
